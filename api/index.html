janitoo_flask.connect_event(event, callback, position='after')[source]¶

Connect a callback to an event. Per default the callback is appended to the end of the handlers but handlers can ask for a higher privilege by setting position to 'before'. Example usage:

def on_before_metadata_assembled(metadata):
    metadata.append('<!-- IM IN UR METADATA -->')
# And in your setup() method do this:
    connect_event('before-metadata-assembled',
                   on_before_metadata_assembled)
janitoo_flask.emit_event(event, *args, **kwargs)[source]¶

Emit a event and return a list of event results. Each called function contributes one item to the returned list. This is equivalent to the following call to iter_listeners():

result = []
for listener in iter_listeners(event):
    result.append(listener(*args, **kwargs))
